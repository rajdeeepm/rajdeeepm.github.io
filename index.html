About me

I’m a senior in Computer Science (B.S.E.) at the University of Michigan with a minor in Mathematics. My current focus is on agentic LLMs, reinforcement learning for tool use, and memory architectures that make models more reliable over long horizons. I love turning research ideas into production‑quality systems that people can actually use — especially in aerospace and automotive settings.

Previously at Oakland University (CS with AI specialization), where I co‑authored multiple IEEE papers on UAV security, datasets, and autonomy.

EECS 485 Projects

As part of UMich’s Web Systems course sequence, I built a series of increasingly sophisticated applications that mirror the evolution of modern web stacks—from static site generation and server‑side interactivity to client‑side applications, distributed computing and search engines.

Web Systems

Templated Static Site Generator

Wrote insta485generator, a Python tool that turns HTML templates, JSON data and static assets into a static Instagram clone. This project honed my skills in templating, HTML/CSS, Python and shell scripting and laid the groundwork for subsequent dynamic applications.

Web Systems

Server‑side Dynamic Pages

Extended the clone with Flask‑powered server‑side pages. Reused templates from P1 and rendered them on demand, implementing full CRUD for users, posts, comments and likes. Learned session management, SQL database integration and built a multi‑user interactive site accessible at localhost:8000.

Web Systems

Client‑side Dynamic Pages

Refactored the server code into a REST API and wrote a JavaScript client to fetch and render data in the browser. Enabled asynchronous features—liking or commenting without reloads, infinite scroll, and double‑click to like—using AJAX and DOM manipulation, demonstrating the power of client‑side apps.

Distributed Systems

Distributed MapReduce Framework

Implemented a MapReduce framework in Python inspired by Google’s paper. Built a Manager that accepts jobs and coordinates multiple Workers over network sockets. Emphasised fault tolerance, concurrency with threads and processes, and demonstrated the framework with a distributed word count across multiple inputs.

Information Retrieval

Scalable Search Engine

Constructed a full‑stack search engine. Created a segmented inverted index using MapReduce with tf‑idf text analysis and PageRank link analysis. Developed an Index server exposing a REST API and a Search UI that returns ranked results—mirroring real web search—with a service‑oriented architecture for scalability.
